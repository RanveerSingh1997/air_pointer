## 0.1.0

- Initial release.
- `CanvasInputSource` abstract boundary interface.
- `CanvasInputController` stream multiplexer.
- `MouseInputSource` for mouse, trackpad, and touch input via Flutter's gesture arena.
- `GestureInputSource` using MediaPipe HandLandmarker (Flutter Web only).
- `OneEuroFilter` for landmark position smoothing.
